smart contract development. We've deployed over 20 sites, 10 custom Discord bots, and 40 smart contracts across
the projects we've helped bring to market. In aggregate, our projects have done over 200m USD in secondary volume and

## Major Repositories

There exist a number of repositories that have been generalized in order to rapidly produce prototypes and MVP's for projects.

### 💸 Budget Collections

The `Budget Collections` repository offers a Solidity smart contract build pipeline, which deploys a generalized form of ERC-721A, and provides the private key of the deployer for further use.

### 🌱 Custom Mint Page

The `Custom Mint Page` repository deploys a mint page using a number of inputs in the form of a `.env` and a templated schema for project files (branding) and smart contract ABI's.

### 🐤 Twitter Sales Bot

The `Twitter Sales Bot` repository is a single, no-change required, script that will run a Twitter NFT Sales Bot (through OpenSea) using a `.env` file to control keys and templating.
